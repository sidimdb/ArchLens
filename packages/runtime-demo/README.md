# runtime-demo

Sample React Native app used to test and demo `@archlens/runtime`.

## Status

**Phase 0 placeholder.** The actual Expo app gets bootstrapped at the
start of Phase 1, once we begin implementing the runtime library.

## Planned contents

A small Expo app with 4–5 screens (Home, Profile, Settings, etc.) that
intentionally contains some UX issues — small touch targets, poor
contrast, missing accessibility labels — so we have realistic things to
annotate during the jury demo.
